My Robot Friend



Project Details

    In this Robot Project I learn:
  1. CSS position to set the positioning of any HTML element top, left, bottom, right - static (default), relative, fixed, sticky, and absolute.
  2. Transition effects add an effect while hover an HTML element (ease, ease-in, ease-out, ease-in-out)
  3. Transform rotate(0 deg) property to rotate robots head, hand, legs.
